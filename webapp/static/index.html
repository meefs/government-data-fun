User documentation

OpenGovDash

A free, no-install web app for exploring 21 US government agencies and ~56 live data endpoints — SEC filings, FDA recalls, BLS employment, NASA, USGS earthquakes, federal spending, campaign finance, and more. Search the raw data yourself, or ask the built-in AI to fetch and cross-reference it for you. Nothing to download. No account. Any API keys you add never leave your browser.

1. Getting started

On first open you'll see a setup wizard with three tracks:

Change tracks or manage keys any time via API Key in the chat top bar. In a hurry? Use "Skip · just show me the data".

2. The AI assistant

The chat can call the government APIs for you. Ask a question; it picks the right agency tools, fetches live data, and answers citing its sources. Tool activity shows inline (⚙ calling query_sec(...) → ↳ query_sec → 8 results). It has one tool per agency plus a cross_reference tool that searches several agencies at once.

Try:

  • "What are the latest SEC 8-K filings that mention cybersecurity?"
  • "Cross-reference 'Samsung' across FCC, FDA, and FTC."
  • "Show recent FDA drug recalls and any related SEC filings."
  • "Search PubMed for GLP-1 agonist clinical trials."
  • "What federal contracts mention 'quantum computing' in USAspending?"

Tips: be specific; the assistant runs up to 5 tool rounds then answers; Claude/OpenAI synthesize better than free models; Groq's free tier is a great fast middle ground.

3. Your API keys are private

Keys are stored only in your browser's localStorage and sent only to that provider. OpenGovDash has no server that sees them. The API Key panel shows a health dashboard (green = verified, amber = rate-limited, red = invalid), live validation as you paste, masked display with a Show toggle, and one-click Revoke.

⚠️ Keys live in the browser — don't enter keys on a shared or public computer.

A free data.gov key unlocks SAM.gov (required there) and raises NASA/FEC rate limits. Optional for everything else.

4. Browsing data yourself

Pick an agency in the left sidebar. Each page has datasets (sub-sections), search where supported, a result limit (10–100) and date range where the API allows it, plus card/table views. Use the Back button (top-left) or your browser's Back — navigation stays inside the app.

6. Troubleshooting

"Free AI hit its shared daily limit" — resets 00:00 UTC. Add a free Groq key for unlimited access.
Agency "temporarily unreachable" — government APIs go down; retry later. NARA/FTC have known upstream outages.
Generic answer / no tool used — free models are weaker at tool decisions; ask directly ("use the SEC tool to…") or switch to Groq/Claude/OpenAI.
Key shows red — check for trailing spaces and that the key matches the provider (OpenAI sk-, Groq gsk_, Gemini AIza, Claude sk-ant-).
SAM.gov "requires a data.gov API key" — expected; get one free at api.data.gov/signup.

7. Privacy

No accounts, no tracking, no analytics. API keys are browser-only and never sent to any OpenGovDash server (there isn't one — it's a static site). Agencies that block browsers are relayed through a thin Cloudflare proxy that strips your auth headers and cookies before forwarding and holds no secrets.
